# PROMPT TAHAP 08 - Laporan Akhir KP

Tahap 8 membangun modul Laporan Akhir KP tanpa mengulang atau merusak modul Tahap 1-7.

Tujuan:
- Mahasiswa dengan assignment aktif/berjalan dapat upload laporan akhir.
- Laporan dapat disimpan sebagai draft dan disubmit untuk review.
- Revisi laporan disimpan sebagai versi baru.
- Pembimbing Dalam mereview laporan mahasiswa bimbingannya.
- Admin/Koordinator memonitor seluruh laporan akhir.
- File laporan disimpan non-public dan diunduh melalui route protected.
- Semua aktivitas penting dicatat dalam log.

Batasan:
- Tidak membuat pengajuan sidang, penjadwalan sidang, penguji, atau penilaian.

Output:
- Migration, model, service, controller, request, route, view, test.
- Report `docs/reports/TAHAP_08_LAPORAN_AKHIR_KP.md`.
- Update spesifikasi dan AGENTS.md.

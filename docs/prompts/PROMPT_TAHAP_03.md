# PROMPT TAHAP 03 - Periode KP, Tempat KP, dan Kuota

Tahap 3 melanjutkan aplikasi SI-KP Farmasi UBP dari Tahap 1 dan Tahap 2 tanpa mengulang setup login, multi-role, dashboard, manajemen user, import, dan profil.

Target utama:
- Membuat modul Periode KP.
- Membuat modul Tempat KP.
- Membuat modul Kuota Tempat KP per periode.
- Membuat log perubahan kuota.
- Membatasi akses modul hanya untuk Admin dan Koordinator KP.
- Menambahkan menu sidebar untuk Periode KP, Tempat KP, Kuota Tempat KP, dan Log Kuota.
- Menambahkan ringkasan Tahap 3 pada dashboard Admin dan Koordinator KP.
- Menyiapkan helper kuota untuk tahap pemilihan tempat/war ticket berikutnya.
- Menambahkan feature test dan dokumentasi Tahap 3.

Batasan tahap ini:
- Belum membuat pendaftaran KP mahasiswa.
- Belum membuat upload dan verifikasi berkas KP.
- Belum membuat pemilihan tempat KP/war ticket.
- Belum membuat daftar tunggu, pembimbing, logbook, laporan akhir, sidang, dan penilaian.

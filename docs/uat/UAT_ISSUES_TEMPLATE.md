# UAT Issues Template

Gunakan format ini untuk mencatat temuan UAT.

| ID | Tanggal | Role | Halaman | Deskripsi Masalah | Langkah Reproduksi | Severity | Status | Catatan Perbaikan | PIC |
|---|---|---|---|---|---|---|---|---|---|
| UAT-001 | YYYY-MM-DD | Admin | /admin/dashboard | Contoh masalah | 1. Login 2. Buka halaman | low/medium/high/blocker | open/in_progress/done/rejected |  |  |

## Severity
- `low`: minor UI/copywriting, tidak menghambat alur.
- `medium`: mengganggu kenyamanan tetapi ada workaround.
- `high`: menghambat fitur penting pada satu role/alur.
- `blocker`: aplikasi/alur utama tidak bisa dipakai.

## Status
- `open`: temuan baru.
- `in_progress`: sedang diperbaiki.
- `done`: sudah diperbaiki dan diverifikasi.
- `rejected`: bukan bug atau tidak masuk scope.
